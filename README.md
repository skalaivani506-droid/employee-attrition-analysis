Machine learning project to predict employee attrition using Logistic Regression
# Employee Attrition Analysis

## Objective
To predict whether an employee is likely to leave the organization using machine learning techniques.

## Dataset
IBM HR Analytics Employee Attrition Dataset.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Model Used
- Logistic Regression

## Results
- Model Accuracy: 88%
- The model performs well in predicting employee retention.
